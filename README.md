# bubble-tip
一个简单的文字提示插件，可以替换原生title提示, 是jquery插件

## 安装
npm install bubble-tip --save

## 使用
```html
    <div style="text-align: center"><span id="test" data-title="我叫你一声大爷你敢答应吗？">我是你大爷</span></div>
```
```js
    $('#test').bubbleTip();
```
